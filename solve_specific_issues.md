### Access mysql via command line on Mac using xampp

```
cd /Applications/XAMMP/xamppfiles/bin
./mysql -u root -p
```

When asked to 'Enter password:', just press enter key.

And, hey presto!

---
Macbook Pro Early 2015 model using El Capitan.
Xampp version 7.0.5-0
